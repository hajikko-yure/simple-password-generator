# 🔐 Password Generator

マスターキー・共通パスワード・サイト名から、安全なパスワードを生成できるツールです！

## 🌟 特徴

- マスターキーは保存しない（記憶で管理OK）
- サイトごとに異なるパスワードを自動生成
- OSSとして自由に使えます！

## 🧠 入力

- マスターキー（自分だけの秘密のキーワード）
- 共通パスワード（覚えやすくてもOK）
- サイト名（例: `Github`）

## 🔐 出力

共通パスワード + 安全なハッシュ値（決定論的に生成）
